# WebBTGadgetBridge
I didn't want the Q-Ring app and Gadget Bridge isn't in Play, so I set this up so I could get telemetry out of a cheap Colmi fitness ring.

- You can use it here: https://rinc3w1nd.github.io/WebBTGadgetBridge/
- You can download it to a device and use it there in a Chrome-based browser with Web Bluetooth enabled.

If you use the HTML direct from your device you might not be able to maintain a persisted state and you are more likely to face cache eviction.

If there is interest in this **make sure to star the repo** so I know. Enough stars and I may make a properly hosted version with optional cloud ZKE-encrypted storage for state persistence so that you, and only you, ever own or even see your data.
